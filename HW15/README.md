# Reflection
Working in a 3D space is both similar in different to working with a 2D one. With a 2D one, I feel like you have to think about things more vertically, and in a 3D one it's more horizontal. But overall it's still positioning things on a canvas with numbers. 

I had some struggles with getting my screen to show anything at all at first, so I was constantly debugging things and couldn't find anything wrong. I'm currently at a hotel, and it turned out that the hotel's wi-fi was blocking/interfering with my live server thing. I switched to my phone hotspot and everything started working after that. 

I was trying to think of a scene that could be easily rendered with only shapes and rotations, so I settled on a solar system. I liked picking out backgrounds to go with my planets. 

I wanted the backgrounds to fade in and out or possibly move a bit, but decided to quit while I was ahead so I didn't break anything. If I were to expand on this, I would replace the colored spheres with actual planet models like the sun has. I would also like to add a subtle glow, but I don't know if that's possible.
